# FileUploaderAPI

FileUploaderAPI is a test web application to upload large files (with a progress bar) to Azure Blob Storage.
 <br />
The next technologies have been used: **ASP.NET Core 8, Angular 17, SignalR, Azure Blob Storage and Email Communication Service**.

[Check the video recording of large file uploading (you need to download a raw file)](https://github.com/petrenko-sergii/FileUploaderAPI/blob/master/docs/LargeFileIsUploaded.avi)
